# 가계부 — 홈 화면 설치용 앱

완전 독립형 오프라인 앱입니다. 인터넷 없이도 열리고, 입력한 모든 데이터는 **그 폰(브라우저) 안에만** 저장됩니다. 서버로 전송되는 것 없음.

## 폴더 안에 든 것
- `index.html` — 앱 본체 (이것만 열어도 동작)
- `manifest.webmanifest` — 앱 정보(이름·아이콘)
- `sw.js` — 오프라인 캐시용
- `icon-192.png`, `icon-512.png`, `apple-touch-icon.png`, `favicon-32.png` — 아이콘

---

## 가장 쉬운 설치 방법 (무료 호스팅 → 홈 화면)

홈 화면 아이콘으로 제대로 쓰려면 이 폴더를 인터넷 주소(URL)에 올려야 해요. 무료로 1분이면 됩니다.

### 1) Netlify Drop (추천, 가입 없이도 가능)
1. 폰이나 PC 브라우저에서 https://app.netlify.com/drop 접속
2. 이 폴더(`ledger-app`) 통째로 드래그해서 업로드
3. 생기는 주소(예: `https://랜덤이름.netlify.app`)를 폰에서 열기

### 2) 홈 화면에 추가
- **아이폰(Safari):** 그 주소 열기 → 공유 버튼 → "홈 화면에 추가"
- **안드로이드(Chrome):** 그 주소 열기 → 우측 상단 ⋮ → "앱 설치" 또는 "홈 화면에 추가"

→ 파스텔 ₩ 아이콘이 홈 화면에 생기고, 누르면 전체화면 앱처럼 떠요. 한 번 연 뒤에는 인터넷 없어도 동작합니다.

## 데이터 관련 주의
- 데이터는 그 브라우저의 저장소에 저장돼요. **같은 주소를 같은 브라우저로** 열면 계속 유지됩니다.
- 브라우저 데이터(캐시·사이트 데이터)를 지우면 입력값도 사라져요.
- 기기를 바꾸면 데이터는 따라가지 않아요(각 기기 로컬 저장).

다른 호스팅(예: GitHub Pages, Vercel)을 써도 됩니다. 폴더 전체를 올리기만 하면 돼요.
